Project Overview

This project is an IoT-based Fire Detection and Monitoring System developed using ESP32 and Firebase. The system monitors temperature, humidity, and fire in real time. Sensors connected to ESP32 send data to Firebase through WiFi. The dashboard displays this data and alerts users during abnormal conditions like fire detection. This system provides a low-cost, real-time safety solution for homes and industries.

Features

  • Real-time temperature monitoring
  • Humidity tracking
  • Fire detection alert system
  • Live dashboard updates
  • Emergency notification

Technologies Used

  • ESP32 (Microcontroller)
  • Firebase Realtime Database
  • HTML, CSS, JavaScript
  • Chart.js
